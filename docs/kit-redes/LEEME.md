# Waraira — Kit de redes (para enviar a quien lleve las redes)

**Para enviar fácil:** manda el archivo **`KIT-REDES-WARAIRA.html`** (un solo archivo, se abre con
doble clic en cualquier computadora, no necesita internet). Adentro están los logos con botón de
descarga + todas las biografías para copiar y pegar.

> Si prefieres mandar todo, comprime esta carpeta `kit-redes/` en un .zip y envíala.

## Qué hay aquí

| Archivo | Para qué |
|---------|----------|
| **KIT-REDES-WARAIRA.html** | **El documento único** (logos descargables + descripciones). Este es el que se envía. |
| `waraira-avatar-400.png` / `-1000.png` | Foto de perfil (avatar) — fondo verde, marca sola. |
| `waraira-marca-1000.png` | Marca sola, fondo transparente — para sobre **fondos oscuros**/fotos. |
| `waraira-marca-verde-1000.png` | Marca sola, fondo transparente — para sobre **fondos claros**. |
| `waraira-portada-1500x500.png` | Portada/banner — X, Facebook, LinkedIn, YouTube. |
| `waraira-logo-oficial.pdf` | Logo oficial en alta (vectorial), para imprenta o el wordmark exacto. |
| `*.svg` | Fuentes editables de cada pieza (para diseño). |

## Reglas que no se rompen
- **Nunca** rostro, nombre ni ubicación de un niño.
- Sin contenido político. No se pide dinero.
- Lema: **«Entre todos, unidos.»** (nunca "a la orden del Estado").
