## 1. Identidad de marca (resumen para quien diseñe los perfiles)

- **Nombre / wordmark:** `waraira` — **en minúsculas**, tipografía **Hanken Grotesk 700**. El punto
  de la "i" es **el faro** (beacon).
- **Significado:** de *Waraira Repano*, nombre caribe de **El Ávila**, la montaña que cuida a Caracas.
  La montaña se alzó cuando una gran ola fue enviada a salvar a las tribus → **imagen protectora para
  una herramienta protectora**.
- **Marca (símbolo):** dos cumbres formando una «w», con un **faro ámbar** en la cima.
- **Principio rector:** *baja barrera para reportar, alta traza para actuar* ("low barrier to report,
  high trace to act").
- **Sensación:** un guardián sereno, cálido y humano. **Sin** banderas ni símbolos políticos.
